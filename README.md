# Intro

A simple visual tool that allows for easily duplicating canvas templates.

As of now, you can include tags by adding a rule for cards with a level one title using the rule name. More is to come but finding a general way to apply it to multiple workflows is a bit challenging, as for my perticular workflow would be easy to implement, but rigid and not verstile for many types.

# Usage

Clicking the ribbon button opens the template generation menu.

![[Screenshot_2026-07-07_09-23-34.png]]

In the settings, you can define rules that will search for a card in the canvas that has that title, it searches for '#' titles.

![[Screenshot_2026-07-07_09-25-22.png]]

If a card exists you can provide default tags for the canvas. This helps when searching your notes, so adding a meta or metadata or tags rule card to your templates is advised.

![[Screenshot_2026-07-07_09-23-56.png]]
